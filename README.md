# Datenbankprojekt zu einem Ferienwohnungsportal 

Dieses Projekt wurde im Rahmen der **Modulteilprüfung** des Moduls **Datenbank- und Informationssysteme 1** im **dritten Semester** als Tandemarbeit erstellt. Ziel war es, ein praxisnahes Datenbankprojekt zu entwickeln und die erlernten SQL-Konzepte umzusetzen.  

## Projektstruktur

- **ER-Modell**:
- Darstellung der Datenbankstruktur und der Beziehungen zwischen Entitäten.  
- **DDL (Data Definition Language)**:  
  SQL-Skripte zum Erstellen und Löschen des Schemas sowie zur Definition von Triggern.  
  - `create_db_schema.sql`  
  - `drop_db_schema.sql`  
- **DML (Data Manipulation Language)**:  
  SQL-Skripte zum Einfügen, Löschen und Verwalten von Daten.  
  - `insert_initial_data.sql`  
  - `delete_initial_data.sql`  

## Zielsetzung

Das Projekt demonstriert die Umsetzung einer relationalen Datenbank inklusive:  
- Datenmodellierung (ER-Modell)  
- Schemaerstellung (DDL)  
- Datenmanipulation (DML)  
- Trigger-Definitionen  

Es dient als praxisnahes Beispiel für die Arbeit mit relationalen Datenbanksystemen.
